# Design System for Food & Beverage E-commerce

## Design Philosophy

### Color Palette
- **Primary**: Warm Terracotta (#D2691E) - Appetizing and inviting
- **Secondary**: Sage Green (#9CAF88) - Natural and fresh
- **Accent**: Cream (#FFF8DC) - Clean and elegant
- **Text**: Charcoal (#2F2F2F) - High contrast readability
- **Background**: Off-white (#FEFEFE) - Clean canvas

### Typography
- **Display Font**: "Playfair Display" - Elegant serif for headings
- **Body Font**: "Inter" - Clean sans-serif for readability
- **Accent Font**: "Dancing Script" - Handwritten style for special elements

### Visual Language
- Modern editorial aesthetic inspired by food magazines
- Clean grid layouts with generous white space
- High-quality food photography with natural lighting
- Subtle shadows and rounded corners for warmth
- Consistent 8px grid system for spacing

## Visual Effects & Styling

### Used Libraries
- **Anime.js**: Smooth micro-interactions and page transitions
- **ECharts.js**: Sales analytics and data visualization
- **Splide.js**: Product image carousels and testimonials
- **Typed.js**: Dynamic text effects for hero section
- **p5.js**: Creative background patterns and animations
- **Pixi.js**: Advanced visual effects for product showcases

### Animation & Effects
- **Hero Section**: Typewriter effect with color cycling text
- **Product Cards**: 3D tilt effect on hover with shadow expansion
- **Image Gallery**: Ken Burns pan/zoom effect on hover
- **Background**: Subtle particle system with food-themed elements
- **Scroll Animations**: Staggered reveal animations for product grids
- **Loading States**: Skeleton screens with shimmer effects

### Header Effect
- Sticky navigation with blur backdrop
- Smooth color transition on scroll
- Shopping cart icon with bounce animation on add
- Search bar with expanding animation

### Interactive Elements
- **Buttons**: Gradient hover states with subtle glow
- **Forms**: Floating labels with focus animations
- **Cards**: Lift effect with depth shadows
- **Images**: Zoom overlay with gradient masks
- **Icons**: Micro-animations for feedback

### Responsive Design
- Mobile-first approach with touch-friendly interactions
- Flexible grid system adapting to screen sizes
- Optimized typography scales for readability
- Gesture-based navigation for mobile users
